总结

  1. NPM 确实可以实现可视化管理，且不暴露 81 端口。
  2. Nginx 在本地，没有必要修改 docker 的网络设置。
  3. 当 docker 的 iptables=false 时，容器的网络需要添加 iptables 规则才能恢复。
  4. Alpine Liunx 的 SSH 需要设置才能使用端口转发。
